Moonstone UI documentation
========================================

Moonstone UI is the open-source front-end for `Cryptonomex Moonstone <https://github.com/cryptonomex/moonstone>`_. A live hosted wallet running a custom themed version of Moonstone UI for Moonstone can be found at `<https://moonstone.openledger.info>`_
